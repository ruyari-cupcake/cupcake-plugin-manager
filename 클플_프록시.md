# 클플 프록시 가이드

로컬 프록시 대신 **Cloudflare Workers 기반 GitHub Copilot CORS Proxy**를 쓰고 싶은 사람이,
자기 Cloudflare 계정에 **그대로 복붙해서 직접 배포**할 수 있게 넣어둔 문서다.

즉 이 문서는:

- 공용 프록시 운영 안내가 아니라
- **각자 자기 계정에 직접 배포해서 쓰는 용도**이고
- 아래 코드를 **복붙해서 Worker에 넣으라는 목적**이다.

---

## 지원 엔드포인트

### Copilot 프록시 (GitHub OAuth 토큰 → tid 자동 교환)
- `/chat/completions`
- `/v1/chat/completions`
- `/v1/messages`
- `/responses`
- `/v1/responses`

### 임베딩 CORS 프록시 (모델명에서 프로바이더 자동 감지)
- `/v1/embeddings`
- `/embeddings`

---

## 배포 방법

1. Cloudflare Dashboard로 이동
2. **Workers & Pages** → **Create Worker**
3. 아래 코드 전체를 **그대로 복붙**하고 Deploy
4. 커스텀 모델 Proxy URL에 Worker URL 입력
   - 예: `https://my-proxy.username.workers.dev`

### Copilot 채팅용
요청 시 `Authorization: Bearer <github_oauth_token>` 또는 `x-api-key` 헤더로 GitHub OAuth 토큰을 보내면,
Worker가 `tid` 토큰을 자동 발급받아 Copilot API로 프록시한다.

### HypaV3 임베딩용
리스AI → 봇 설정 → 하이파V3:
- 모델: **custom**
- Custom Server URL: **`https://my-proxy.username.workers.dev/v1`**  (본인 워커 URL + /v1)
- API Key: **본인의 임베딩 API 키** (Voyage, OpenAI 등)
- Model: **본인이 쓰는 모델명** (voyage-3, text-embedding-3-small 등)

모델명에서 프로바이더가 자동 감지됨:
| 모델명 | 프로바이더 |
|---|---|
| `voyage-*` | Voyage AI |
| `text-embedding-*` | OpenAI |
| `embed-*` | Cohere |
| `jina-*` | Jina |
| `mistral-*` | Mistral |

## Worker 코드

```js
// ══════════════════════════════════════════════════════════
// GitHub Copilot CORS Proxy + 임베딩 프록시 — Cloudflare Workers
// ══════════════════════════════════════════════════════════
// 지원 엔드포인트:
//   Copilot: /chat/completions, /v1/messages, /responses
//   임베딩:  /v1/embeddings (모델명에서 프로바이더 자동 감지)
//
// 사용법:
//   1. Cloudflare Dashboard → Workers & Pages → Create Worker
//   2. 이 코드 전체를 붙여넣기 → Deploy
//   3. Copilot: 커스텀 모델 Proxy URL에 워커 URL 입력
//      예: https://my-proxy.username.workers.dev
//   4. 임베딩: HypaV3 Custom Server URL에 워커 URL + /v1
//      예: https://my-proxy.username.workers.dev/v1
//      API Key에 본인의 임베딩 API 키, Model에 모델명 입력
// ══════════════════════════════════════════════════════════

const COPILOT_TOKEN_URL = "https://api.github.com/copilot_internal/v2/token";
const COPILOT_API_BASE = "https://api.githubcopilot.com";
const CHAT_VERSION = "0.26.2025030601";
const CODE_VERSION = "1.99.2025030601";
const USER_AGENT = `GitHubCopilotChat/${CHAT_VERSION}`;

// Copilot 경로 목록
const COPILOT_PATHS = [
  "/chat/completions",
  "/v1/chat/completions",
  "/v1/messages",
  "/responses",
  "/v1/responses",
];

// 임베딩 경로
const EMBEDDING_PATHS = ["/v1/embeddings", "/embeddings"];

// 모든 허용 경로
const ALLOWED_PATHS = [...COPILOT_PATHS, ...EMBEDDING_PATHS];

// ── 임베딩 모델명 → API URL 자동 매핑 ──
const EMBEDDING_ROUTES = [
  { test: (m) => m.startsWith("voyage"), url: "https://api.voyageai.com/v1/embeddings", name: "Voyage AI" },
  { test: (m) => m.startsWith("text-embedding"), url: "https://api.openai.com/v1/embeddings", name: "OpenAI" },
  { test: (m) => m.startsWith("embed-"), url: "https://api.cohere.com/v2/embed", name: "Cohere" },
  { test: (m) => m.startsWith("jina"), url: "https://api.jina.ai/v1/embeddings", name: "Jina" },
  { test: (m) => m.startsWith("mistral"), url: "https://api.mistral.ai/v1/embeddings", name: "Mistral" },
];

function detectEmbeddingUrl(modelName) {
  if (!modelName) return null;
  const m = modelName.toLowerCase().trim();
  for (const route of EMBEDDING_ROUTES) {
    if (route.test(m)) return { url: route.url, name: route.name };
  }
  return null;
}

function generateHexId(length = 64) {
  const arr = new Uint8Array(length / 2);
  crypto.getRandomValues(arr);
  return Array.from(arr, (b) => b.toString(16).padStart(2, "0")).join("");
}

// ── tid 토큰 발급 ──
async function refreshTidToken(apiKey) {
  const resp = await fetch(COPILOT_TOKEN_URL, {
    method: "GET",
    headers: {
      Accept: "application/json",
      Authorization: `Bearer ${apiKey}`,
      "User-Agent": USER_AGENT,
    },
  });

  if (!resp.ok) {
    const text = await resp.text();
    throw new Error(`tid token failed (${resp.status}): ${text}`);
  }

  const data = await resp.json();
  if (!data.token || !data.expires_at) {
    throw new Error("tid token response missing token or expires_at");
  }

  return data.token;
}

// ── CORS 헤더 ──
const CORS_HEADERS = {
  "Access-Control-Allow-Origin": "*",
  "Access-Control-Allow-Methods": "GET, POST, PUT, DELETE, OPTIONS",
  "Access-Control-Allow-Headers": "*",
  "Access-Control-Max-Age": "86400",
};

// ── JSON 응답 헬퍼 ──
function jsonResp(status, data) {
  return new Response(JSON.stringify(data), {
    status,
    headers: { "Content-Type": "application/json", ...CORS_HEADERS },
  });
}

// ── 경로 정규화: /v1/chat/completions → /chat/completions 등 ──
function normalizePath(pathname) {
  if (pathname === "/v1/chat/completions") return "/chat/completions";
  if (pathname === "/v1/responses") return "/responses";
  return pathname;
}

// ── 임베딩 CORS 프록시 핸들러 ──
async function handleEmbeddingProxy(request) {
  const rawBody = await request.text();

  // body에서 model 추출
  let modelName = "";
  try {
    const parsed = JSON.parse(rawBody);
    modelName = (parsed.model || "").toString().trim();
  } catch { /* 파싱 실패해도 계속 */ }

  // 모델명으로 프로바이더 자동 감지
  const detected = detectEmbeddingUrl(modelName);
  if (!detected) {
    return jsonResp(400, {
      error: `모델명 "${modelName}"에서 프로바이더를 자동 감지할 수 없습니다.`,
      supported: EMBEDDING_ROUTES.map((r) => `${r.name}: ${r.url}`),
      hint: "지원되는 모델명 접두사: voyage-*, text-embedding-*, embed-*, jina-*, mistral-*",
    });
  }

  // 클라이언트의 Authorization 그대로 전달 (Copilot tid 교환 안 함)
  const forwardHeaders = {
    "Content-Type": request.headers.get("Content-Type") || "application/json",
  };
  const auth = request.headers.get("Authorization");
  if (auth) forwardHeaders["Authorization"] = auth;
  const xApiKey = request.headers.get("x-api-key");
  if (xApiKey) forwardHeaders["x-api-key"] = xApiKey;

  const targetResp = await fetch(detected.url, {
    method: "POST",
    headers: forwardHeaders,
    body: rawBody,
  });

  return new Response(targetResp.body, {
    status: targetResp.status,
    headers: {
      "Content-Type": targetResp.headers.get("Content-Type") || "application/json",
      ...CORS_HEADERS,
    },
  });
}

// ── 메인 핸들러 ──
export default {
  async fetch(request) {
    // preflight
    if (request.method === "OPTIONS") {
      return new Response(null, { status: 204, headers: CORS_HEADERS });
    }

    const url = new URL(request.url);

    // health check
    if (url.pathname === "/" || url.pathname === "/health") {
      return jsonResp(200, {
        status: "ok",
        copilot_endpoints: COPILOT_PATHS,
        embedding_endpoints: EMBEDDING_PATHS,
        embedding_auto_detect: EMBEDDING_ROUTES.map((r) => `${r.name}: ${r.url}`),
        usage: {
          copilot: "Authorization: Bearer <github_oauth_token>",
          embedding: "Authorization: Bearer <embedding_api_key> + model명으로 프로바이더 자동 감지",
        },
      });
    }

    // ── 임베딩 경로 → 별도 핸들러 (Copilot tid 교환 안 함) ──
    if (EMBEDDING_PATHS.includes(url.pathname)) {
      if (request.method !== "POST") return jsonResp(405, { error: "Method Not Allowed" });
      try {
        return await handleEmbeddingProxy(request);
      } catch (e) {
        return jsonResp(502, { error: e.message });
      }
    }

    // 경로 확인
    const normalizedPath = normalizePath(url.pathname);
    if (!COPILOT_PATHS.includes(normalizedPath) && !COPILOT_PATHS.includes(url.pathname)) {
      return jsonResp(404, { error: "Not Found", allowed: ALLOWED_PATHS });
    }

    if (request.method !== "POST") {
      return jsonResp(405, { error: "Method Not Allowed" });
    }

    try {
      // ── API 키 추출 (Authorization 또는 x-api-key) ──
      const authHeader = request.headers.get("Authorization") || "";
      const xApiKey = request.headers.get("x-api-key") || "";
      const apiKey = authHeader.startsWith("Bearer ")
        ? authHeader.slice(7)
        : xApiKey;

      if (!apiKey) {
        return jsonResp(401, { error: "API key required (Authorization: Bearer <token> or x-api-key header)" });
      }

      // ── tid 토큰 발급 ──
      let tidToken;
      try {
        tidToken = await refreshTidToken(apiKey);
      } catch (e) {
        return jsonResp(401, { error: e.message });
      }

      // ── 요청 본문 ──
      const rawBody = await request.text();
      const hasVision =
        rawBody.includes('"image_url"') ||
        rawBody.includes('"image"') ||
        rawBody.includes('"input_image"');
      const isStream =
        rawBody.includes('"stream":true') ||
        rawBody.includes('"stream": true');

      // ── UUID 생성 ──
      const sessionId = crypto.randomUUID() + Date.now().toString();
      const interactionId = crypto.randomUUID();
      const requestId = crypto.randomUUID();

      // ── Copilot 대상 URL 결정 ──
      const targetPath = normalizePath(url.pathname);
      const copilotUrl = `${COPILOT_API_BASE}${targetPath}`;

      // ── Copilot 공통 헤더 ──
      const copilotHeaders = {
        Authorization: `Bearer ${tidToken}`,
        "Content-Type": "application/json",
        "Copilot-Integration-Id": "vscode-chat",
        "Editor-plugin-version": `copilot-chat/${CHAT_VERSION}`,
        "Editor-version": `vscode/${CODE_VERSION}`,
        "User-Agent": USER_AGENT,
        "Vscode-Machineid": generateHexId(64),
        "Vscode-Sessionid": sessionId,
        "X-Github-Api-Version": "2025-10-01",
        "X-Initiator": "user",
        "X-Interaction-Id": interactionId,
        "X-Interaction-Type": "conversation-panel",
        "X-Request-Id": requestId,
        "X-Vscode-User-Agent-Library-Version": "electron-fetch",
      };

      // /v1/messages (Anthropic) → anthropic-version 헤더 추가
      if (targetPath === "/v1/messages") {
        copilotHeaders["anthropic-version"] = "2023-06-01";
        const clientAnthropicVersion = request.headers.get("anthropic-version");
        if (clientAnthropicVersion) {
          copilotHeaders["anthropic-version"] = clientAnthropicVersion;
        }
      }

      // 비전 요청
      if (hasVision) {
        copilotHeaders["Copilot-Vision-Request"] = "true";
      }

      // ── Copilot API 호출 ──
      const copilotResp = await fetch(copilotUrl, {
        method: "POST",
        headers: copilotHeaders,
        body: rawBody,
      });

      // ── 응답 패스스루 + CORS 헤더 추가 ──
      const responseHeaders = {
        "Content-Type":
          copilotResp.headers.get("Content-Type") || "application/json",
        ...CORS_HEADERS,
      };

      if (isStream) {
        responseHeaders["Cache-Control"] = "no-cache";
        responseHeaders["Connection"] = "keep-alive";
      }

      return new Response(copilotResp.body, {
        status: copilotResp.status,
        headers: responseHeaders,
      });
    } catch (e) {
      return jsonResp(500, { error: e.message });
    }
  },
};
```
